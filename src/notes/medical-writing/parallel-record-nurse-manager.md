# 被看见的实习生

> 本文为临床实习期间完成的一篇平行病历，记录一次关于医护同事关系、临床团队氛围与职业温度的观察。

刚进入病区实习时，我常常觉得自己像一个临时停靠在科室里的人。

每天早上晨交班前，我会提前到示教室吃早饭。示教室里人来人往，护士们准备交班，医生们翻看病历，而我坐在角落里，尽量安静地吃完早饭，等着一天的工作开始。

那时候我对“同事关系”的理解其实很简单：医生负责诊疗，护士负责护理，大家按照各自的岗位完成工作，尽量配合、不出差错，就已经是一种好的关系。作为实习生，我更习惯把自己放在一个很小的位置上——少说话，多观察，不给老师添麻烦。可是杨老师的出现，让我慢慢意识到，临床团队里的关系并不只是靠职责和流程维系的，也靠一些很细微、很日常的善意支撑着。

第一次让我有这种感觉，是在一个普通的早晨。我照常坐在示教室吃饭，杨老师主动和我说话，还把麻薯分给我。后来有一次，她把一个鸡蛋递给我，笑着说自己胆固醇有点高，只能吃一个，另一个就给我吃。她说得很自然，像是生活里再寻常不过的一句话。但对当时的我来说，这并不是一个鸡蛋那么简单。

在科室里，实习生有时是很容易“隐形”的。我们穿着白大褂，却还没有真正独立承担临床工作的能力；我们每天跟在老师身后，却又不完全属于任何一个固定的团队。很多时候，我会下意识提醒自己不要占用太多空间，不要麻烦别人。可是杨老师递过来的那个鸡蛋，让我突然感觉自己不是一个只需要默默旁听和完成任务的实习生，而是一个也会被注意、被照顾的人。

后来，杨老师给在座的护士们分香片，也顺手给了我一片。那片香片很小，小到如果只是记录事件，似乎没有什么值得写的。但也正是这样的小事，让我感受到一种“被包括在内”的温暖。她没有刻意区分谁是护士、谁是医生、谁只是短暂停留的实习生。她只是很自然地把我也放进了这个早晨的示教室里，放进了这个科室共同生活的一部分里。

还有一次，我把胸牌落在了示教室。杨老师发现后提醒我收好。胸牌只是一个小物件，却象征着一个人在医院里的身份。对患者来说，胸牌让他们知道眼前的人是谁；对医务人员来说，胸牌也提醒我们自己正在承担怎样的角色。杨老师的提醒让我想到，临床工作中的许多安全感，正是来自这些不动声色的彼此照应。一个人被提醒，并不一定意味着做错了什么，也可能意味着有人在帮你守住那个还不够熟练的职业边界。

最让我印象深刻的，是一次接开水时发生的小意外。

那天我不小心烫到了手，第一反应是赶紧找凉水冲。可是女厕所外面的水是温的，只有男厕所外面的水比较凉。我站在那里冲手的时候，心里其实有点尴尬，也有点不知所措。烫伤并不算严重，可是在陌生的临床环境里，一点小意外也会让实习生突然意识到自己的笨拙。

杨老师看见后，马上问我怎么了。她知道我被烫到后，没有大惊小怪，也没有责备我不小心，而是很快让我去拿冰袋敷一下。她问我知不知道冰袋在哪里，我摇头，她就直接带我去拿。这个过程很短，短到在忙碌的病区里几乎可以被忽略。但那一刻，我感受到的不是简单的帮助，而是一种临床环境中非常珍贵的稳定感：当一个人遇到状况时，有人会看见，有人会停下来，有人知道下一步该怎么做，并愿意带你走到那里。

医学里常常谈“照护”。过去我总以为，照护主要发生在医务人员和患者之间。医生诊断疾病，护士执行护理，患者接受治疗，这是最典型的照护关系。可是杨老师让我看到，照护也可以发生在同事之间，发生在师生之间，发生在一个团队内部。她对我的关心并不因为我只是实习生而减少，也不因为事情很小就被忽略。她让我意识到，一个真正温暖的临床团队，不只是对患者专业负责，也会让团队中的每个人感到安全和被尊重。

更难得的是，这种温和并不是偶然的。无论是晨交班时，还是平时和下属护士交流时，杨老师大多都语气温和、面带笑容,甚至称呼我们实习同学为“老师”。她安排工作时清楚有条理，提醒问题时也很少让人感到难堪。作为护士长，她不仅在管理一个护理团队，也在用自己的方式塑造这个病区的气质。

一个科室的氛围，有时并不是写在制度里的，而是藏在每天早晨的语气里，藏在交班时的表情里，藏在一个人愿不愿意多问一句“你怎么了”里。

这段经历让我重新理解了“同事关系”。临床中的同事关系，不只是医嘱与执行，不只是分工与协作，也不只是工作流程上的衔接。更深一层看，它是一种彼此托举：在高压、忙碌、责任重的环境里，大家仍然愿意看见彼此作为“人”的处境。有人紧张，有人疲惫，有人初来乍到，有人不小心受了伤。而好的团队，会让这些时刻不至于变成孤立无援的时刻。

杨老师给我的并不只是麻薯、鸡蛋、香片，或者一个冰袋。她给我的是一种关于临床工作的提醒：医学人文并不只存在于病床旁，也存在于医护之间的日常相处里。一个人怎样对待同事，怎样对待实习生，怎样处理那些看似不重要的小事，其实都在体现她对职业的理解。

以后我也会成为临床团队中的一员。也许那时我会面对更复杂的病情、更繁忙的工作和更大的压力。但我希望自己还能记得这些早晨：示教室里被递来的鸡蛋，落下胸牌时温和的提醒，还有烫伤后被带去拿冰袋的那几分钟。它们让我知道，临床工作当然需要技术、知识和判断，但同样需要一种能看见他人的能力。

在医院里，温暖并不一定轰轰烈烈。它可能只是晨交班前的一句问候，是把另一个鸡蛋递给角落里的实习生，是看见别人不知所措时停下来问一句：“怎么了？”

而正是这些细小的瞬间，让我在忙碌的病区里感受到：医学不只是治病，也是在人与人之间不断学习如何彼此照顾。
